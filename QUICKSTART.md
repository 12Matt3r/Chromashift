## ðŸ› Troubleshooting

### Problem: Black Screen After Loading
**Solution**: 
- Check browser console (F12) for errors
- Ensure WebGL is enabled in your browser
- Try a different browser (Chrome recommended)

### Problem: Images Not Loading
**Solution**:
- Ensure `imgs/` folder is in same directory as `index.html`
- Check that image files exist: `supermarket_dreamscape_7.png`, etc.
- If images still fail, the system will generate a procedural panorama

### Problem: Choice Bubbles Don't Respond
**Solution**:
- Make sure JavaScript is enabled
- Check that `chromashift.js` loaded properly (browser console)
- Try refreshing the page (F5)

### Problem: Panorama Won't Rotate
**Solution**:
- Click and drag on the panorama area
- Ensure you're not clicking UI elements
- Try using mouse instead of trackpad

### Problem: Low Performance / Laggy
**Solution**:
- Close other browser tabs
- Update your graphics drivers
- Lower browser zoom to 100%
- Use a device with better GPU
